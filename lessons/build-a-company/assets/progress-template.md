# Progress template

Use this format when saving learner progress to memory. Save as a `project` type memory with name `build-a-company-progress`.

```markdown
---
name: build-a-company-progress
description: Learner progress through the build-a-company skill
type: project
---

## Learning path

1. [completed/in-progress/pending] Topic filename — brief topic description
2. [completed/in-progress/pending] Topic filename — brief topic description
3. [completed/in-progress/pending] Topic filename — brief topic description
4. [completed/in-progress/pending] Topic filename — brief topic description
5. [completed/in-progress/pending] Topic filename — brief topic description

## Current position

- Last completed topic: [filename or "none"]
- Last concept covered: [brief description]
- Next up: [filename and topic]

## Key takeaways so far

- [Bullet point for each completed topic's main takeaway]

## Notes

- [Anything relevant about the learner's engagement, questions, or interests that should inform future sessions]
```
